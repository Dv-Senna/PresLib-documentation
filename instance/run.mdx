# `pl::Instance<API>.run(...)`

## Definitions

```cpp
inline void run();
```

Run the presentation.

## How it works

This function is the mainloop of the presentation. It handles event, render blocks, choose which slide to render and will handle animations in the future. Note that this function blocks the execution of the program. This function must be call in the main thread.
